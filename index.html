My bachelor's degree thesis. Together with 2 of my colleagues, we created an online multiplayer monopoly-like board game.

The majority of the game's logic has been implemented in a PostgreSQL relational database. We have also created a front-end React application for users to interact with the game.

My part was focused on working with an API (C#) that connected the client app with our database, as well as the final E2E testing of the game utilising Selenium solutions.

This version of our game can be found here. A new version is in development, run solely by one of our thesis-group members.


A desktop application which serves as a tool for scraping on-sale game listings published on Valve's Steam platform.

The app was written in Java, and makes use of the Jsoup HTML parser as well as Google's Gson serialization / deserialization library. Equipped with a GUI made with JavaFX & SceneBuilder it provides a fluent user experience.

It allows for viewing game listings' discount percentages (functionality recently restricted by Valve), listings' core information, sorting listings by prices / title / discount as well as visiting listings' individual store pages.

The application has been released for Windows, Mac & Linux and is available to download here.
